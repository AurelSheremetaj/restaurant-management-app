# Restaurant Point of Sale System (Work in Progress)

## Description
This project is a desktop Point of Sale (POS) application for restaurant management.
It supports multiple user roles and provides functionality for managing menus,
tables, and customer orders.

## Implemented Features
- Role-based access (Admin, Waiter, Cook)
- Menu and menu item management
- Table order creation and management
- Order summary with VAT calculation
- Invoice preview
- Modular UI using Kivy/KivyMD
- Database layer prepared (PostgreSQL)

## Work in Progress
Some modules are currently implemented as UI placeholders.
Additional database integration and feature completion are planned.

## Technologies
- Python
- Kivy / KivyMD
- PostgreSQL (psycopg2)
